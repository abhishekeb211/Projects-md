## Verification Command Summary

### Quick Verification Script

Create `verify-compliance.sh`:

```bash
#!/bin/bash
# Phase 7.1: Venue Compliance Verification Script

echo "=== IEEE TSE Compliance Check ==="
echo "Word count:"
texcount -merge -sum aegiscli-paper.tex 2>/dev/null || echo "texcount not installed"

echo -e "\nFigure format check:"
find . -name "*.png" -o -name "*.jpg" | grep -v node_modules | wc -l

echo -e "\nCitation count:"
grep -c "\\\\cite" aegiscli-paper.tex sections/*.tex 2>/dev/null | awk '{s+=$1} END {print s}'

echo -e "\nAnonymization check:"
grep -i "github\|gitlab" aegiscli-paper.tex sections/*.tex | grep -v "todo\|Zenodo" | wc -l

echo -e "\n=== ACM TOSEM Compliance Check ==="
echo "Author metadata:"
grep -A 5 "\\author" aegiscli-paper.tex

echo -e "\n=== ICSE SEIP Compliance Check ==="
echo "Page count (requires PDF):"
echo "Run: pdfinfo aegiscli-paper-final.pdf | grep Pages"
```
