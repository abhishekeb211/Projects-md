### Package Assembly Script

**Script:** `create-tse-package.sh`

```bash
#!/bin/bash
set -euo pipefail

PACKAGE_NAME="SUBMISSION-PACKAGE-TSE"
PACKAGE_DIR="${PACKAGE_NAME}"

echo "Creating IEEE TSE submission package..."

# 1. Create package directory
rm -rf "${PACKAGE_DIR}"
mkdir -p "${PACKAGE_DIR}"

# 2. Copy main PDF (anonymized)
if [ -f "aegiscli-paper-final.pdf" ]; then
    cp aegiscli-paper-final.pdf "${PACKAGE_DIR}/"
    echo "✅ Copied main PDF"
else
    echo "⚠️ Warning: aegiscli-paper-final.pdf not found"
fi

# 3. Copy cover letter
if [ -f "cover-letter.pdf" ]; then
    cp cover-letter.pdf "${PACKAGE_DIR}/"
    echo "✅ Copied cover letter"
else
    echo "⚠️ Warning: cover-letter.pdf not found"
fi

# 4. Create supplemental materials ZIP
if [ -d "supplemental-materials" ]; then
    cd supplemental-materials
    zip -r "../${PACKAGE_DIR}/supplemental-materials.zip" .
    cd ..
    echo "✅ Created supplemental-materials.zip"
else
    echo "⚠️ Warning: supplemental-materials/ directory not found"
fi

# 5. Copy artifact DOI
if [ -f "artifact-doi.txt" ]; then
    cp artifact-doi.txt "${PACKAGE_DIR}/"
    echo "✅ Copied artifact DOI"
else
    echo "⚠️ Warning: artifact-doi.txt not found"
fi

# 6. Create README
cat > "${PACKAGE_DIR}/README-SUBMISSION.txt" << EOF
IEEE TSE Submission Package
===========================

Contents:
1. aegiscli-paper-final.pdf - Main manuscript (anonymized)
2. cover-letter.pdf - Cover letter addressing novelty and methodology
3. supplemental-materials.zip - Appendices, protocols, schemas
4. artifact-doi.txt - Zenodo DOI for artifact

Package created: $(date)
EOF

# 7. Calculate checksums
cd "${PACKAGE_DIR}"
md5sum *.pdf *.zip *.txt > checksums.md5 2>/dev/null || md5 *.pdf *.zip *.txt > checksums.md5 2>/dev/null || echo "Checksum calculation skipped (md5sum/md5 not available)"
cd ..

# 8. Create TAR.GZ archive
tar -czf "${PACKAGE_NAME}.tar.gz" "${PACKAGE_DIR}"

echo "✅ Package created: ${PACKAGE_NAME}.tar.gz"
echo "Package size: $(du -h ${PACKAGE_NAME}.tar.gz | cut -f1)"
echo "Contents:"
ls -lh "${PACKAGE_DIR}/"
```

**Usage:**
```bash
chmod +x create-tse-package.sh
./create-tse-package.sh
```
