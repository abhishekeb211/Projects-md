## Custom Commands

The `macros.tex` file defines custom commands for:

### Placeholder Commands

- `\result{exp}{TBD}` - Marks experimental results that need to be filled (e.g., `\result{E1}{TBD}`)
- `\todo[inline]{description}` - Marks TODO items requiring action (uses red text for visibility)

### Code Formatting

- `\code{text}` - Formats inline code using `\texttt` (e.g., `\code{aegiscli scan}`)

### Mathematical Notation

- `\finding` - Finding tuple `f` in mathematical notation
- `\policy` - Policy function `π` in mathematical notation  
- `\triage` - Triage function `τ` in mathematical notation
- `\mttr` - Mean Time To Remediate abbreviation

### Acronym Commands

- `\sast`, `\dast` - Static/Dynamic Application Security Testing
- `\sarif` - Static Analysis Results Interchange Format
- `\pac` - Policy-as-Code
- `\opa` - Open Policy Agent
- `\llm` - Large Language Model

All acronyms use `\textsc{}` for small caps formatting, consistent with IEEE style.
