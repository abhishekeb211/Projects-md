# Literature Cards Directory

This directory contains structured literature cards for all 50 included papers from the SLR.

## Structure

- `C1_orchestration/` - Papers on scanner orchestration & SARIF normalization (10-12 papers)
- `C2_ai_triage/` - Papers on AI/LLM in security analysis (10-12 papers)
- `C3_pac/` - Papers on Policy-as-Code & security debt (8-10 papers)
- `C4_adoption/` - Papers on DevSecOps adoption & security champions (8-10 papers)
- `C5_privacy/` - Papers on privacy-preserving AI & air-gap security (8-10 papers)

## Card Naming Convention

Format: `[Cluster]_[Sequence]_[AuthorYear]_[TitleKeyword].md`

Example: `C1_001_Smith2020_SARIFAdoption.md`
